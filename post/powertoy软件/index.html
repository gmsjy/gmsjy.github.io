简介

PowerToys是微软出品的一组开源实用程序，可以帮助用户调整和简化Windows的使用体验。

安装和运行PowerToys

安装条件

1.电脑操作系统Windows10 v1903(build 18362)之后的新系统。 2.需要.NET CORE 3.1 Desktop Runtime运行软件。

PowerToys下载安装

从GitHub网站上下载PowerToys安装包。

按照提示一步步安装即可。 安装完成后，如下图所示目前最新版本是v0.31.1。

功能介绍

截止现在PowerToys提供了8个系统增强的小工具，下面一次介绍。

Color Picker(颜色选择器)

使用Win+Shift+C可以激活工具在屏幕任意处获取屏幕颜色值，在下边的选择器行为里面可以设置粘贴板内颜色的格式。

FancyZones

FancyZones这个功能是在桌面上分割不同的区域，用于在多任务不同窗口协作时使用。

File Explorer add-ons(文件资源管理器)

文件资源管理器为文件资源管理器添加了Markdown和SVG两种类型文件的预览功能。

Image Resize(图像大小调整)

图片大小调整用于快速调整图片大小，集成在鼠标右键上方便使用。图片调整结果可以再设置页面根据自己的需求自行设置。

KeyBoard Manager(键盘管理器)

键盘管理器可以方便快捷的重新配置快捷键。

PowerRename(强力重命名)

PowerRename集成在右键的一个文件重命名工具，其强大之处在于支持正则表达式，完全满足日常文件重命名功能。

PowerToys Run(PowerToys启动器)

PowerToys Run是一个结合了Wox和Winddows Walk的快速启动器，通过Alt+Space唤醒。输入关键词可以实现搜索和程序启动，快速高效。

Shortcut Guide(快捷键指南)

长按Win键具体时间在设置中可以自行设置，会浮现Windows快捷键的指示图标。
